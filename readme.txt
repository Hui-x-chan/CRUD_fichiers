Un projet GitHub collaboratif

1) créer un projet "CRUD_fichiers"
2) le chef de projet le rend versionnable (avec git)
3) création d'un fichier README où il explique le but du projet
4) le chef de projet va le placer sur GitHub
5) le chef de projet ajoute les collaborateurs du projet
6) Web Designer et le Dev PHP vont cloner sur leur poste le projet
7) Web Designer : css/style.css + index.html (branche webdesign)
8) Dev PHP : index.php (branche PHP)
9) le chef de projet traite les changements
